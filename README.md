# Randevu Sistemi — Site Oluşturucu

Basit bir yönetim paneli üzerinden firma bilgileri girilerek özelleştirilmiş randevu web siteleri oluşturulmasını sağlayan full-stack bir uygulama.

## Özellikler

- Dashboard üzerinden site oluşturma ve yönetme
- Özelleştirilebilir firma profilleri
- Dinamik randevu sistemi
- Responsive müşteri sitesi
- Netlify Functions desteği
- Kalıcı veri saklama desteği
- TypeScript tabanlı backend mimarisi
